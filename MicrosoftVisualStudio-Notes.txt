View: To add a view , add a new folder 'Game'
Add - New Item - .NET Core - MVC View Page.
Create (ei: index.cshtml, and Create.cshtml)>

Controllers:

  select .NET CORE, and select MVC Controller Class..
   Name: GameController.cs

Create a new Folder : Models , in the same tree as View and Controllers..

  Right click Models folder, and then select Add (Left pane 'Code', right pane "Class").
  Name : Game.cs
